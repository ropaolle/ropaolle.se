Wordpress theme - Hemma hos ropaolle

# Add Beans
``` bash
# Setup
cd /opt/bitnami/apps/wordpress/htdocs/wp-content/themes
sudo chown daemon:ropaolle ropaolle/
cd ropaolle
git init
git remote add origin git@github.com:ropaolle/ropaolle.se.git
git remote -v

# Fetch
git fetch origin
```
